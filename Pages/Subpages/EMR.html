Placeholder text for the EMR project page. This will be a way to test the app I am devloping for work and a way for users to see the results. I will provide a csv file that they can view the faux EMR data on, and have a javascript application to visualize as the application progresses.

The application has multiple

  1. Loading and building run configurations, early fail for invalid inputs or possible runtime errors. This step creates a job id object and will return that with first a 200 good response

  3. The final step will be a cleaned and paramterized version of the data ready for database insertion. Database transaction will not be part of this display.
