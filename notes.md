    * Relative positioning
        Using position: relative;
            - creates coordinate system within a container element
            - put this in the parent
        Using position: absolute;
            - When a child of a container using position: relative; this command finds the nearest parent with the command position: relative;
            - Properties ( top,bottom,left,right )
        Positioning Coordinates
            - Negative values push outside the boundaries
            - top: 0 would align the top edge of the child with the top edge of the parent, top: -3rem would put it 48 pixels above on standard
